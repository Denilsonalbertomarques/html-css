A propriedade all do css é utilizada para resetar os valores de todas as propriedades que um seletor recebe. 

            

                Exemplo: <div class="child">
                
                .child {

                    background-color:black;
                    border:3px solid white;
                    color:white;
                    all: initial;
                }
Isso significa que a tag que tem a class child não terá nenhuma das propriedades que foi definida.

a propriedade all também pode ser utilizada para que um elemento filho herde todos os valores de seu elemento pai. E também para resetar todos os valores que o elemento filho recebeu de seu pai. 

            

                Exemplo: <div class="container">

                    <div class="filho"></div>

                </div>

                    .container {
                        background-color:black;
                        border:3px solid white;
                        color:white;
                        padding:20px;
                    }
                    .filho {
                        all:inherit;
                    }
Isso significa que o elemnto que tem a class filho herdará todas os valores do elemento que tem a class container, por ser seu pai

OBS: por padrão os filhos herdam de seu pai algumas propriedades como: color e font-size podemos alterar isso utilizando a propriedade all, ou podemos alterar, outros valores para color e font-size
